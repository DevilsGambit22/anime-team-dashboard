Owner guide

Updating the dashboard

  1. Edit links, playlist entries, and optional manual events in config.js.
  2. Place audio files inside assets/music/.
  3. Upload the project to a GitHub repository and enable GitHub Pages.
  4. Replace the old Chess.com iframe URL with your new GitHub Pages URL.

The Chess.com match calendar does not require Supabase. It requests the public club-match endpoint directly from the browser.
